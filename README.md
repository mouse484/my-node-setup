# sao-my-node-setup

> my praiseworthy SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao my-node-setup my-project
```

### From git

```bash
sao mouse_484/sao-my-node-setup my-project
```
